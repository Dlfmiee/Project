# 🔥 Fire Hand: AI Gesture Superpower

A real-time computer vision project that uses MediaPipe and OpenCV to detect hand gestures and trigger a premium fire particle effect.

## ✨ Features
- **Real-time Hand Tracking:** Fast and accurate detection using MediaPipe.
- **Dynamic Particle System:** Procedural fire with color gradients and smoke effects.
- **Smooth Transitions:** Fire intensity fades in and out gracefully.
- **Interactive UI:** On-screen status dashboard and FPS counter.
- **Sound Ready:** Pre-built logic for immersive audio feedback.

## 🛠️ Setup & Installation
1. **Clone/Download** this repository.
2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the App:**
   ```bash
   python fire_hand.py
   ```

## 🎮 How to Use
- **Ignite:** Open your hand fully (5 fingers extended).
- **Extinguish:** Close your hand into a fist.
- **Quit:** Press `q` on your keyboard.

## 📂 Project Structure
- `fire_hand.py`: Main application logic.
- `requirements.txt`: Library dependencies.
- `steps_to_wow.md`: Tips for maximizing the visual experience.
